RMS System (Reability Management System)

The purpose of this document is to define Standard Operating Procedures as they apply to the onsemi Reliability Management System. This document will apply to anyone using ONRMS for requesting global reliability services, tracking reliability progress, documenting reliability input and output and determining reliability results. 1.2The scope of the onsemi Reliability Management System (ONRMS) is to support tracking of products processed in the Global ONsemi Reliability Labs.

LRT System (Live Reability Transaction)

During the reliability process, most important item is a data accuracy but one of common issue in order to get it smooth is human error. Recognizing the need to achieve less human dependent procedure, the SBN Rel Lab Engineering team has designed and developed a Live Reliability Transaction system (LRT) for reliability production. By using barcode system through barcode scanning method instead of typing in ONRMS system able to eliminate human errors in every process.
